# CV React Static

Sitio estático tipo CV construido con React y Vite.

## Requisitos

- Node.js 18 o superior
- npm

## Instalación

1. Instala dependencias:

```bash
npm install
```

2. Levanta el servidor local:

```bash
npm run dev
```

3. Abre la URL que te muestre Vite, normalmente:

```bash
http://localhost:5173
```

## Build de producción

```bash
npm run build
```

## Vista previa del build

```bash
npm run preview
```

## Estructura

- `src/App.jsx`: composición principal del CV
- `src/styles.css`: estilos y animaciones
- `src/main.jsx`: punto de entrada de React
- `src/profile.png`: foto de perfil
